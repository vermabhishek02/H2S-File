Border Surveillance Drone System A full-stack web-based surveillance system designed to monitor border areas using drone live feed, real-time alerts, and secure backend communication.

Project Overview The Border Surveillance Drone System is built to:

Stream live video from a drone camera
Detect suspicious activity (optional AI integration)
Display real-time alerts
Provide a secure dashboard for monitoring
Store logs and surveillance data
This project demonstrates full-stack development using frontend, backend, and real-time communication technologies.
